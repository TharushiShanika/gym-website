Gym Promotional Website
A modern and responsive promotional website for a gym, built with React and Vite.

🔧 Tech Stack
Frontend: React, TypeScript, Vite

Styling: CSS / Tailwind CSS (if used)

Deployment: GitHub Pages

⚙️ Setup Instructions
Clone the repository:
bash git clone https://github.com/TharushiShanika/gym-website.git 
cd gym-website# 

Getting Started
Run npm install
Run npm run dev
