# Gym Promotional Website

A modern and responsive promotional website for a gym, built with React and Vite.  

## 🔧 Tech Stack

- **Frontend:** React, TypeScript, Vite  
- **Styling:** CSS / Tailwind CSS (if used)  
- **Deployment:** GitHub Pages  

## ⚙️ Setup Instructions

1. Clone the repository:

bash
git clone https://github.com/TharushiShanika/gym-website.git
cd gym-website# Magic Patterns - Vite Template

## Getting Started

1. Run `npm install`
2. Run `npm run dev`
